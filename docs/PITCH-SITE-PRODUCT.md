## The Problem With Pitch Decks

Every day, thousands of founders, consultants, and nonprofit leaders send pitch decks that get opened once — maybe — skimmed for 90 seconds, and forgotten. The format is broken:

- **Slides kill narrative momentum.** Discrete pages break the emotional arc.
- **PDFs are passive.** There's no soundtrack, no animation, no immersion.
- **They require you to be there.** Without the presenter, the deck is a skeleton.
- **Friction kills delivery.** "Can you open this .pptx?" "Which version of Keynote?" "The fonts are wrong."

The result? Your big idea arrives as a commodity — indistinguishable from the 50 other decks in the recipient's inbox.

---

## The Solution: A Self-Presenting Pitch Site

We build single-file, self-contained pitch experiences that present themselves. No installs. No accounts. No dependencies. One HTML file that works everywhere — email it, AirDrop it, host it, or hand it over on a USB stick.

### How It Works

The recipient opens the file and sees two options:

- **Play Presentation** — Sit back mode. Narration starts, the site auto-scrolls through each section, and the full story unfolds hands-free. At the end, a call-to-action pulses and a contact modal opens.
- **Browse Freely** — Explore at your own pace like a normal website. Every section uses scroll-triggered animations that replay naturally as you navigate.

That's it. No onboarding. No instructions. Just an experience.

## What's Inside

### Cinematic Scrollytelling
Full-viewport sections with parallax backgrounds, scroll-triggered reveal animations, and animated stat counters that build emotional momentum continuously — not in disconnected slides.

### Auto-Narrated Presentation Mode
Professional voiceover synchronized to each section. The pitch delivers at full impact whether you're in the room or not. Scroll is locked during playback so the experience is controlled and intentional.

### Bilingual by Default
One-click language toggle (ES/EN or any language pair). Every heading, paragraph, button, and narration track switches instantly. Essential for cross-border pitches.

### Self-Contained Single File
Everything — images, fonts, scripts, styles — embedded in one HTML file. No external dependencies, no broken links, no "the images didn't load." Works offline. Works forever.

### Interactive Elements
- Species/product cards with lightbox zoom
- Animated timeline roadmaps
- Stat counters that replay on every visit
- Contact modal with download CTA

### Presentation Flow Control
- Scroll lock during auto-play (no accidental navigation)
- Exit button always available (never trap the user)
- Post-presentation reset (overlay returns for a fresh start)
- CTA pulse animation draws the eye to the next step

## Technical Architecture

Built for durability and zero-maintenance:

- **No build step** — pure HTML/CSS/JS, no frameworks, no node_modules
- **No backend** — static file, deployable anywhere (Vercel, Netlify, S3, email attachment)
- **No dependencies** — Google Fonts loaded at runtime, everything else embedded
- **Progressive enhancement** — works without JavaScript (static content), enhanced with it (animations, narration)
- **Mobile responsive** — adapts to any screen size
- **Performance** — single HTTP request, no waterfall loading
